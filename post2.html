Keamanan Cloud untuk Blog Pribadi

19 April 2025

Keamanan adalah aspek penting dalam mengelola blog di lingkungan cloud. Berikut adalah beberapa tips keamanan yang saya terapkan:

Keamanan Jaringan

Menggunakan Security Group untuk mengontrol akses ke instance EC2. Saya hanya mengizinkan SSH dari IP publik saya, dan HTTP dari mana saja.

Keamanan Data

Menggunakan Amazon S3 dengan konfigurasi yang tepat untuk menyimpan backup blog saya. Bucket dikonfigurasi untuk diakses hanya oleh saya (private access), dan mengaktifkan enkripsi sisi server.

Update dan Patch

Menjaga sistem operasi dan perangkat lunak server web tetap diperbarui adalah hal yang sangat penting. Saya secara teratur menjalankan perintah `yum update` untuk memperbarui paket pada Amazon Linux AMI.
